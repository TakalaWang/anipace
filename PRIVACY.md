# 隱私權政策 / Privacy Policy

**AniPace**

最後更新:2026-07-10

AniPace 不收集、不儲存、不傳送任何使用者資料。

- 不收集個人資訊、瀏覽紀錄或使用行為。
- 不使用 cookie、追蹤器或分析工具。
- 不與任何第三方或伺服器連線。
- 所有程式碼都在本機執行,僅在動畫瘋 (ani.gamer.com.tw) 頁面上偵測並重設影片播放速度。

有任何問題請至 https://github.com/TakalaWang/anipace/issues 提出。

---

AniPace does not collect, store, or transmit any user data. It uses no cookies,
trackers, or analytics, and makes no network requests. All code runs locally and
only adjusts video playback speed on ani.gamer.com.tw.

Questions: https://github.com/TakalaWang/anipace/issues
