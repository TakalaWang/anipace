<h1 align="center">AniPace・1 倍速警察</h1>

<p align="center">看動畫就應該用 1 倍速觀看。</p>

---

在 [動畫瘋](https://ani.gamer.com.tw)(`ani.gamer.com.tw`)上,把播放速度強制維持在 1 倍速。

## 安裝

### 開發者模式(手動載入)

1. 下載 / `git clone` 這個 repo
2. Chrome 開 `chrome://extensions`
3. 右上角開「開發者模式」
4. 「載入未封裝項目」→ 選這個資料夾

### Chrome Web Store

審核通過後補上連結。
