# Chrome Web Store 送審清單

到 https://chrome.google.com/webstore/devconsole →「新增項目」→ 上傳 `anipace.zip`,然後照下面填。

## 商店資訊 (Store listing)

**名稱**
```
AniPace
```

**簡短說明**(上限 132 字)
```
在動畫瘋強制 1 倍速觀看。誰把影片調成非 1 倍速,就自動拉回並提醒你:看動畫就應該用 1 倍速觀看。
```

**詳細說明**
```
AniPace 只做一件事:在巴哈姆特動畫瘋 (ani.gamer.com.tw) 上,把影片播放速度強制維持在 1 倍速。

只要你(或任何腳本)把速度調成非 1 倍速,AniPace 會立刻把它拉回 1 倍速,並跳出一行通知:
「看動畫就應該用 1 倍速觀看」。

• 只在動畫瘋生效,不碰 Netflix、YouTube 或其他任何網站
• 不要求任何額外權限,不收集任何資料
• 全螢幕也看得到通知
• 開源:https://github.com/TakalaWang/anipace
```

**類別**:娛樂
**語言**:中文(繁體)

**圖示**:zip 內已含 128×128(`icon128.png`)
**螢幕截圖**:上傳 `store-screenshot.png`(1280×800)

## 隱私權 (Privacy) 分頁

**單一用途說明**
```
在巴哈姆特動畫瘋 (ani.gamer.com.tw) 將影片播放速度強制維持在 1 倍速。
```

**權限理由** — 若被問到 `ani.gamer.com.tw` 主機權限:
```
需要在動畫瘋頁面注入內容腳本,以監控並在使用者變速時將播放速度重設為 1 倍速。
```

**資料使用**:全部勾選「不收集」,並勾選底部三項認證(不販售資料、不作無關用途、不用於信用評估)。
**遠端程式碼**:選「否,不使用遠端程式碼」(所有程式碼都包在套件內)。

## 發布

發布範圍可選「公開」或限「台灣」。填完按送審,通常 1–3 天。
